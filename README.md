# 📊 Bank Loan Analysis using Python

## 📌 Project Overview

This project analyzes a bank loan dataset to uncover lending patterns, loan performance, customer characteristics, and financial insights. Using Python for data cleaning, analysis, and visualization, the project helps identify trends in loan applications, funded amounts, repayments, interest rates, and loan quality.

The objective is to transform raw financial data into meaningful business insights that can support better lending decisions and risk assessment.

---

## 🎯 Project Objectives

* Analyze overall bank loan performance.
* Evaluate funded amounts and repayment trends.
* Identify Good Loans vs Bad Loans.
* Analyze customer demographics and loan characteristics.
* Discover monthly lending trends.
* Generate business insights through data visualization.

## 🚀 How to Run the Project

1. Clone this repository.

```bash
git clone https://github.com/your-username/Bank-Loan-Analysis.git
```

2. Navigate to the project directory.

```bash
cd Bank-Loan-Analysis
```

3. Install the required libraries.

```bash
pip install pandas numpy matplotlib seaborn openpyxl
```

4. Open the Jupyter Notebook.

```bash
jupyter notebook
```

5. Run all cells to reproduce the analysis.
